
PITBULL FRONTEND (Vercel-ready)

Quick start:
1. npm install
2. npm run dev
3. Open http://localhost:5173

Environment:
- VITE_API_BASE -> backend URL (example: https://pitbull-backend.onrender.com)
- VITE_ADMIN_PIN -> 4444

Deployment to Vercel:
- Build Command: npm run build
- Output Directory: dist
- Set env vars VITE_API_BASE and VITE_ADMIN_PIN in Vercel dashboard
